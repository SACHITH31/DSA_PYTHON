### Core Array Operations & Complexity

| Operation              | Time  |
| ---------------------- | ----- |
| Access by index        | O(1)  |
| Traverse               | O(n)  |
| Insert at end          | O(1)* |
| Insert at start/middle | O(n)  |
| Delete                 | O(n)  |
